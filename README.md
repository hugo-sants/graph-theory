# Graph Theory

This repository contains implementations and exercises related to Graph Theory, developed as part of academic activities in the Computer Science program at the Federal University of Campina Grande (UFCG).

## Objective

The main goal of this repository is to explore and practice concepts of Graph Theory, including:

- Representation of graphs using appropriate data structures
- Analysis of graph properties
- Implementation of graph-based algorithms
- Solving practical problems involving networks

## Covered Topics

### Graph Concepts

- Directed and undirected graphs  
- Weighted and unweighted graphs  
- Bipartite graphs  
- Graph connectivity  
- Induced subgraphs

## Academic Context

This repository was developed as part of coursework and practical assignments, focusing on applying theoretical concepts of Graph Theory to practical problems.
